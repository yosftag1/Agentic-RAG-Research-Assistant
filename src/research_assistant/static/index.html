About This Assistant

This system is a multi-agent retrieval-augmented research assistant built for iterative evidence-based work.

How It Works

  • Orchestrator: classifies your request and decides which agents to run.
  • Retriever: searches your knowledge base first and expands to web sources only when local context is not enough.
  • Summarizer / Analyst / Writer: synthesize findings, compare evidence, and draft structured responses.
  • Merge step: produces the final answer with source-backed citations.

Model Flexibility

You can switch between providers (Gemini, OpenAI, Ollama). Embedding changes automatically reset stale vectors so the index stays consistent.

If you use Ollama from a remote or hosted environment, you may need a tunnel (for example, ngrok) so this app can reach your local Ollama endpoint.

Transparency

The live run panel shows what each agent is doing, and the pipeline panel exposes detailed step-by-step reasoning metadata.
